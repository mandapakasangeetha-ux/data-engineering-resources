# Data Engineering Resources

A lightweight repository of data engineering learning materials, setup references, and architecture visuals.

## Repository Overview

This repo is organized for learners and practitioners to quickly access essential tools, training guides, and reference diagrams for data engineering.

## Contents

- `README.md` - Project overview and guidance for using the repository.
- `tree.md` - Current repository structure in tree format.
- `software-list` - Minimum recommended tools for Day 1 setup.
- `training-session-2026` - Initial session note.
- `training-modules/` - Topic-based training materials.
  - `architecture-data-engineering/` - Architecture diagrams and cloud platform visuals.
  - `basic-git-cheetsheet-20260621/` - Git session notes and reference PDF.
  - `databricks/` - Placeholder for Databricks resources.
  - `linux/` - Placeholder for Linux training content.
  - `shell-scripting/` - Placeholder for shell scripting tutorials.
  - `software-to-be-installed/` - Placeholder for software installation guides.
  - `sql-advance/` - Placeholder for advanced SQL resources.

## Getting Started

1. Open `software-list` and install the recommended tools.
2. View the architecture diagrams under `training-modules/architecture-data-engineering/`.
3. Read the Git session PDF under `training-modules/basic-git-cheetsheet-20260621/`.
4. Expand the placeholder folders with content for Databricks, Linux, shell scripting, software installation, and SQL.

## Recommended Tools

The `software-list` includes a Day 1 setup checklist:

- AWS CLI
